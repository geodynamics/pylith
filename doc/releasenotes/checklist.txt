TAG

  * Tag source code.

  git tag -a vX.X.X -m "MY MESSAGE"
  git push origin vX.X.X

  To delete a tag:
    REMOTE: git push --delete origin vX.X.X
    LOCAL: git tag -d vX.X.X

  Updating a tag label
    git tag new old
    git tag -d old
    git push origin :refs/tags/old
    git push --tags
